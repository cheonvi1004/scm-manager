Get all

URL /api/rest/groups(.xml|.json)
Method GET
Returns 200 OK
401 Unauthorized
415 Unsupported Media Type

JSON Example

GET /api/rest/groups.json

{ groups: [
  {
    name: 'group1',
    members: [
      'hans', 'klaus', 'uwe'
    ]
  },
  {
    name: 'group2',
    members: [
      'peter', 'klaus', 'hans'
    ]
  }
]}
